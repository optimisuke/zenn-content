# はじめに

3Dプリンター持ってる方に聞きたいんですが、ロボットアーム作りたくなりません？なりますよね？自分はなります。買ったときから「いつかアームを…」とずっと思ってました。

今回作ったのは、micro:bit + ring:bit2 + 3Dプリントのパーツで動くロボットアームです。micro:bit を傾けると加速度センサーでアームが動き、ボタンでグリッパーを開閉できます。ハマったことや工夫したことをまとめておきます。

![完成動画](/images/2026-05-08-robotarm.gif)

全然ロボットアームとしての役割を果たせていないんですが、動きがおもしろいのでヨシッ！
![](/images/2026-05-08-15-12-27.png)

# 使用したもの

## ハードウェア

- **3Dプリンター**: Bambu Lab A1 mini
- **サーボ**: MS18（SG90 互換）× 3
- **マイコン**: micro:bit v2
- **拡張ボード**: Elecfreaks ring:bit2
- **接着剤**: LOCTITE ピンポインター ゼリー状 瞬間接着剤

## 開発環境

- [micro:bit Python Editor](https://python.microbit.org/v/3) — ブラウザで MicroPython を書いて書き込めるやつ

## 3Dモデル

- [Servo Mini Arm](https://makerworld.com/ja/models/1290483-servo-mini-arm#profileId-1320674)
- [Servo Mini Arm V2 Cyberbrick](https://makerworld.com/ja/models/1455332-servo-mini-arm-v2-cyberbrick)

今回は V2 の方を使いました。違いはよく把握できていないんですが…。

「Cyberbrick」は Bambu が展開しているおもちゃ向けの電子工作プラットフォームで、3Dプリントしたパーツと組み合わせて動くものが作れるというコンセプトです。面白い試みだと思うので、日本でも入手しやすくなってほしいところです。

# 経緯

最初は手元にあった **M5StickC Plus** とサーボ用拡張ボードを使おうとしていたんですが、内蔵の充電池が死んでいたため断念。ちょうど **micro:bit v2 と ring:bit2** が手元にあったので、そちらに切り替えました。

# 構成

ring:bit2 はサーボポートが P0 / P1 / P2 の 3 つ。

| ポート | 役割               |
| ------ | ------------------ |
| P0     | アーム（第1関節）  |
| P1     | アーム（第2関節）  |
| P2     | グリッパー（開閉） |

---

# 工夫したこと

## 組み立て前に 90 度で固定する

サーボは電源を入れていない状態では手で動かせてしまいます。パーツを取り付ける前にサーボが何度を向いているかわからないまま組むと、完成後にズレが生じます。

今回は動作確認用の別プログラム（`servo_test.py`）を用意して、**組み立て前にすべてのサーボを 90 度に固定してからパーツを取り付けました**。これで可動範囲が対称になり、後の調整がかなり楽になりました。

## 操作方法の設計

サーボ 3 軸 + 操作インターフェースの組み合わせをどうするか悩みましたが、最終的にこういう方針にしました：

- **アーム 2 軸（P0 / P1）→ 加速度センサー**  
  micro:bit を傾けるだけで直感的に操作できます。左右傾きで第1関節、前後傾きで第2関節。

- **グリッパー（P2）→ A / B ボタン**  
  アームを傾けながら同時にボタンで開閉できるので、片手で操作できます。

# ハマったこと

## 台座が安定しない

台座部分が固定できず、アームが動くたびに倒れたり外れたりしてしまう問題が発生しました。アーム全体の重量に対して台座が軽すぎることが原因と思われます。

ただ、台座がないまま動かしてみたら、動きが魚っぽくなって面白かったです。ちゃんと固定されていないからこそ、虫とか動物っぽい動きになる。これはこれでアリかもしれない。

## 水平回転をあきらめた

水平方向の回転（ベース回転）を追加したかったんですが、ring:bit2 のサーボポートは 3 つまで。第1・第2関節とグリッパーで使いきってしまうため断念しました。

水平回転を追加するには、より多くのサーボポートを持つ拡張ボードへの変更が必要です。

## 固定方法

本来はネジで固定するつもりでしたが、ちょうどよいサイズが見つからず断念。パーツ同士の固定は **LOCTITE ピンポインター ゼリー状 瞬間接着剤**だけにしました。ゼリー状なので垂直面や隙間のある箇所にも塗りやすく、3Dプリント品の固定に適していました。

# 操作方法

## 動作確認用（servo_test.py）

| 操作     | 動作                     |
| -------- | ------------------------ |
| A ボタン | サーボを選択             |
| B ボタン | 選択中のサーボをスイープ |
| A+B 同時 | 全サーボ同時スイープ     |

## メインプログラム（main.py）

| 操作                     | 動作                                    |
| ------------------------ | --------------------------------------- |
| micro:bit を左右に傾ける | P0（第1関節）                           |
| micro:bit を前後に傾ける | P1（第2関節）                           |
| A ボタン                 | P2 グリッパー 閉じる                    |
| B ボタン                 | P2 グリッパー 開く                      |
| A+B 同時                 | 全サーボ 90度（ニュートラル）にリセット |

# プログラム

## 動作確認用（servo_test.py）

組み立て前の準備と動作確認のために使いました。具体的には、サーボを 90 度に固定してからパーツを取り付けること、サーボが壊れていないかの確認、ring:bit2 でサーボをどう動かすかの感触をつかむこと、の 3 つが目的です。

```python
from microbit import *

PINS = [pin0, pin1, pin2]

for p in PINS:
    p.set_analog_period(20)

def angle_to_analog(deg):
    deg = max(0, min(180, deg))
    return int(26 + (deg / 180) * 102)

def set_angle(pin, deg):
    pin.write_analog(angle_to_analog(deg))

def sweep(pin, label):
    display.scroll(label, delay=60)
    for deg in range(0, 181, 10):
        set_angle(pin, deg)
        sleep(80)
    for deg in range(180, -1, -10):
        set_angle(pin, deg)
        sleep(80)
    set_angle(pin, 90)

for p in PINS:
    set_angle(p, 90)
display.show(Image.YES)
sleep(500)

servo_index = 0
labels = ["P0", "P1", "P2"]
display.show(str(servo_index))

while True:
    if button_a.is_pressed() and button_b.is_pressed():
        display.show("A")
        for deg in range(0, 181, 10):
            for p in PINS: set_angle(p, deg)
            sleep(80)
        for deg in range(180, -1, -10):
            for p in PINS: set_angle(p, deg)
            sleep(80)
        for p in PINS: set_angle(p, 90)
        display.show(str(servo_index))
    elif button_a.is_pressed():
        servo_index = (servo_index + 1) % 3
        display.show(str(servo_index))
        sleep(300)
    elif button_b.is_pressed():
        sweep(PINS[servo_index], labels[servo_index])
        display.show(str(servo_index))
        sleep(300)
    sleep(50)
```

## メインプログラム（main.py）

加速度センサーでアーム 2 軸を制御、A/B ボタンでグリッパーを開閉します。

```python
from microbit import *

PINS = [pin0, pin1, pin2]

for p in PINS:
    p.set_analog_period(20)

def angle_to_analog(deg):
    deg = max(0, min(180, deg))
    return int(26 + (deg / 180) * 102)

def set_angle(pin, deg):
    pin.write_analog(angle_to_analog(deg))

def accel_to_angle(val):
    val = max(-1024, min(1024, val))
    return int((val + 1024) / 2048 * 180)

all_angles = [90, 90, 90]
for i, p in enumerate(PINS):
    set_angle(p, all_angles[i])

display.show(Image.YES)
sleep(500)

while True:
    if button_a.is_pressed() and button_b.is_pressed():
        all_angles = [90, 90, 90]
        display.show("R")
    else:
        all_angles[0] = accel_to_angle(accelerometer.get_x())
        all_angles[1] = accel_to_angle(accelerometer.get_y())
        if button_a.is_pressed():
            all_angles[2] = max(0, all_angles[2] - 5)
        elif button_b.is_pressed():
            all_angles[2] = min(180, all_angles[2] + 5)

    for i, p in enumerate(PINS):
        set_angle(p, all_angles[i])
    sleep(50)
```
