# はじめに

3Dプリンター(Bambu Lab A1 mini)を買って、4ヶ月経ちました。今使っているのは2台目の3Dプリンターで、**1台目は印刷の失敗が多くて、いつの間にか使わなくなって**しまってました。

今回買う際、最近の機種はむちゃくちゃ簡単と聞いていたのですが、実際その通りでした。おかげで**失敗も少なく、順調に使い続けられて**います。このことは次のセクションで詳しく書きます。

「3Dプリンターは気になるけど、実際何に使うのか想像しづらい、使い続けられるか自信ない」という人は多いんじゃないかと思います。私も同じように心配していたのですが、この4ヶ月で印刷したものを数えたら、**成功208回、モデルの種類は113種類**で、いい感じに継続できています。同じように気になってる人向けに、実際に何を作ってきたかをカテゴリー別にまとめてみます。

# なぜ今回は続けられているのか

1台目を中途半端にやめてしまった反省もあるので、続けられてる理由を整理しておきます。

一番は、**単純に失敗が少ないこと**です。1台目は失敗のたびに気力が削られていって、それが触らなくなった一番の原因でした。

次に大きいのが、Bambu Labの3Dモデル共有サービス「[MakerWorld](https://makerworld.com/)」の存在です。いろんな人が投稿してるモデルを眺めて、気に入ったものをそのままスマホから印刷できます。**「作りたい」から「印刷開始」までのハードルが低い**ので、無料で公開してくれてる作者の方々には日々感謝です。

自分でモデリングするときも、CAD?([build123d](https://github.com/gumyr/build123d))のコードを最近はAIに書いてもらえるので、**寸法さえ測れば「ここをこう直したい」と言うだけで済む**ようになりました。ちなみに普段はソフトウェアエンジニアをしていて、CADは触ったことがなかったので、この身軽さにはかなり助けられてます。

あと、A1 mini自体がかなり安い機種（いまだと26,000円。やばい。）で、もっと高い機種の購入も考えていたので、 **「高い機種はあきらめたからフィラメントは贅沢してもいいか」** という気持ちになれてます。なので **「フィラメント大量に買っちゃってるし、これもプリントしとくか」** となってます。ただ数えたら4ヶ月で15個買っていて、フィラメント代が本体価格を超えてきてるので、もう少し考えて買おうとは思ってます。。。

# 何を作ってきたか

Bambu Studioの印刷履歴を見返して、成功した208回分をカテゴリー分けしました。キャンセル・失敗も22回ありましたが(サポート材の設計ミスとか、たまにあるスパゲッティ化とか)、そのへんは今回割愛して、うまくいったものだけ載せます。

大半はMakerWorldからのダウンロードですが、一部だけ自分でモデリングしたものもあります。数が多いものや見た目が面白いものを、カテゴリーごとに厳選して紹介します。

# 数字で振り返る4ヶ月

- **印刷成功: 208回** / 失敗・キャンセル: 22回
- **作ったモデルの種類: 113種類**
- **一番作ったジャンル: 収納・整理整頓(49種類・95回)** → 生活の困りごと解決が中心でした
- 一番印刷回数が多かった単一モデル: カスタマイズ可能なクレート(11回、サイズ違いで量産)

体感としても、フィギュアやおもちゃより **「暮らしのちょっとした困りごとを埋める」使い方が圧倒的に多かった** です。ネットに落ちてるモデルをそのまま印刷するだけでも十分楽しめるし、サイズだけ変えて自分の環境にぴったり合わせられるのが既製品にはない強みだと思ってます。

# おわりに

4ヶ月使ってみて、**家の中の「なんかちょうどいいのがない」ことの全てが印刷対象になる**感じです。「気になってるけど何に使うか分からない」という人に、こういう使い方もあるんだと伝わったら嬉しいです。

**「また使わなくなるんじゃないか」という不安は、4ヶ月続いた今はだいぶ薄れました**。今回はキャプチャ画像のサムネイルだけの紹介になったので、次は実際に印刷した現物の写真も交えて、使い勝手とか失敗談も含めて書いてみたいなと思ってます。

これから買おうか迷ってる人や、**1台目を使わなくなってしまった人の参考になれば**嬉しいです。
